Guía para hacer un Render de HTML en Django

1 - Crear CARPETA con el nombre "templates" dentro de la carpeta del proyecto (Dentro de esta carpeta van los archivos con extensión .html)

2 - Vamos al archivo SETTINGS.PY y indicamos donde se ubiva nuestro directorio del proyecto y de la carpeta "templates" que creamos. Lo hacemos dentro de la LISTA de nombre TEMPLATES y queda así:

     'DIRS': [BASE_DIR,'templates'],

3 - Vamos al archivos VIEWS.PY e importamos el modulo RENDER de esta forma (al inicio y sin comillas): "from django.shortcuts import render". Así Django podrá renderizar los archivos HTML

2 - Dentro de la carpeta "templates" crear un archivo con el nombre que elijamos con extensión .HTML (Ejemplo: hola.html)

3 - Vamos al archivo VIEWS.PY y creamos la funcion para usar en el archivo HTML. Algo asi: DEF nombredelafunción(request) y debajo correctamente identado: return render(request, 'hola.html')

4 - vamos al archivos URLS.PY y agregamos el path o ruta de nuestra función creada recien. Ejemplo: path('hola/', views.nombredelafunción)

5 - Vamos a la terminal y levantamos el servidor de pruebas de Django con este comando sin comillas: "python manage.py runserver"

6 - vamos a nuestro explorador web (Chrome o Firefox o Edge) y escribimos http://127.0.0.1:8000/ y damos enter

7 - Si todo funciona bien debería aparece un listado de urls, empezando por ADMIN y luego ver HOLA.

8 - Tipeamos en la barra de direcciones http://127.0.0.1:8000/hola y listo, debería mostrar el mensaje que pusimos en la función que hicimos en el archivo VIEWS.PY
